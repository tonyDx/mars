# 背景

文档定义HTML书写的基本规范，目的为增强团队合作，提高代码质量。

## 基本规则

### 缩进

使用四个空格代表缩进，不混用tab和space

### 字母

仅限使用小写字母，适用于HTML标签，特性，特性值，属性及属性值。

### 编码

使用UFT-8

## HTML 规范

### 文档类型

使用`HTML5`, `<!DOCTYPE html>`。

自闭合标签不封闭。

```html
<!--good-->
<br>

<!--bad-->
<br />

```

### 语义

语义化使用HTML标签，段落元素就应该用`p`，标题就应该用`h1`等。

### 优雅退级

多媒体内容应该提供优雅的退级，最常见的如`img`，应该添加`alt`，提供图片加载失败的文字说明。

### 分离

严格保持结构，样式和行为的分离。
